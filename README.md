# Birthday Invitation Microsite

A cinematic mobile-first birthday invitation experience featuring a luxury emerald green, ivory, sage, and champagne gold palette.

## Features

- Mystery landing screen
- 6-card luxury memory game with 15-second timer
- Floating cake challenge: tap 3 cakes in 10 seconds
- Cinematic emerald-and-gold vault unlock animation
- Personalized invitation reveal card

## Technologies Used

- HTML5
- CSS3 (Glassmorphism, animations)
- JavaScript (ES6+)

## Setup

1. Clone the repository
2. Open `index.html` in a modern web browser
3. For full experience, add the following audio files to `assets/` folder:
   - `ambient-piano.mp3`: Soft piano background music (looping)
   - `crystal.mp3`: Crystal UI sound effect
   - `unlock.mp3`: Vault unlock sound effect

   If audio files are not present, the experience will work without sound.
